Frequently Asked Questions

Apa itu HTML?
HTML adalah singkatan dari Hypertext Markup Language. Ini adalah bahasa markup yang digunakan untuk membuat struktur dasar dari halaman web.
Apa itu CSS?
CSS adalah singkatan dari Cascading Style Sheets. Ini adalah bahasa yang digunakan untuk mengatur tampilan dan presentasi elemen-elemen yang telah didefinisikan menggunakan HTML.
Apa itu JavaScript?
JavaScript adalah bahasa pemrograman yang digunakan untuk membuat interaktivitas pada halaman web.
